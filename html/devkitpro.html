Installing devkitPro on an existing Arch Linux install

Written on 10/6/2021

Setting Env Variables

DEVKITPRO=/opt/devkitpro
DEVKITARM=/opt/devkitpro/devkitARM
DEVKITPPC=/opt/devkitpro/devkitPPC

Importing the Keys

sudo pacman-key --recv BC26F752D25B92CE272E0F44F7FD5492264BB9D0 --keyserver keyserver.ubuntu.com
sudo pacman-key --lsign BC26F752D25B92CE272E0F44F7FD5492264BB9D0

Downloading and Installing the Keyring

wget https://pkg.devkitpro.org/devkitpro-keyring.pkg.tar.xz

After running this, install the keyring via this command

sudo pacman -U devkitpro-keyring.pkg.tar.xz

Edit /etc/pacman.conf & add the following lines

[dkp-libs]
Server = https://pkg.devkitpro.org/packages

[dkp-linux]
Server = https://pkg.devkitpro.org/packages/linux/$arch/

After following the last step, resync the database and update installed packages

sudo pacman -Syu

Congratulations! You have installed devkitPro on your Arch Install!

You can now download the development packages via pacman, ex:

sudo pacman -S 3ds-dev
